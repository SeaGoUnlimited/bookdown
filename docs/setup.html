Chapter 2 Accessing the Database with Docker

Some info about docker

2.1 Instructions for Mac/Linux users

  1. Open command terminal
    • Shortcut: Command+space, type “terminal”
  2. Login to Docker account in the terminal
  3. Pull docker image:
  4. Run image:
  5. View container IDs and copy the most recently created one:
  6. Use “docker cp” to copy the database to a local folder:

$ docker cp containerID:/spotter.sqlite/Users/mollywilliams/Database

Nice job! Jump to Chapter 3 to learn how to navigate the database using SQLite Studio.

2.2 Instructions for Windows users

  1. Go to your GitHub account: github.com/username
  2. Click on Repositories, and the green button “New” to create a new repo
  3. Name this new repo “awesome-book”
  4. DO NOT initialize this repo with a README
  5. Click the green “create repository” button — this will take you to your new repo
  6. Click the green “clone or download” button and COPY URL
